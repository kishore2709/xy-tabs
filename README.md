# xy-tabs

---

> 基于`React Hooks` + `typescript`的基础组件

## 安装

```sh
# npm
npm install --save xy-tabs

# yarn
yarn add xy-tabs
```

## 使用

```ts
import React from "react";
import ReactDOM from "react-dom";
import XyTabs from "xy-tabs";
ReactDOM.render(<XyTabs />, container);
```

## 开发

```sh
yarn run start
```

## 例子

http://localhost:6006

## 测试

```
yarn run test
```
